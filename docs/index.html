Integration with Sandman

Mockzure is designed to work seamlessly with Sandman for local development and testing:

  • Configure Sandman to use Mockzure as the Azure endpoint
  • Use the OAuth2/OIDC endpoints for authentication testing
  • Test VM management operations without affecting real Azure resources
  • Simulate different user roles and permissions
